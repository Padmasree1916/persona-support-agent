# Dashboard loading and browser cache

If the dashboard remains blank, first check the Adsparkx status page. Then open a private window. If that works, clear site data for `app.adsparkx.example`, disable content-blocking extensions for the site, and sign in again. Supported browsers are the latest two versions of Chrome, Edge, Firefox, and Safari.

Do not clear all saved browser passwords. If the issue continues, capture the browser version, timestamp, workspace ID, and a screenshot of the developer console without exposing tokens.
